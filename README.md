# 社团管理系统 java1063

## 项目概述

社团管理系统是一款基于前后端分离架构的软件，旨在为社团的日常管理和运营提供便捷高效的解决方案。

## 技术栈

- 前端：Vue、Element-UI
- 后端：Spring Boot、MyBatis Plus
- 数据库：MySQL

## 功能模块

### 管理员后台

- 登录
- 用户管理
- 社团管理
- 社团类型管理
- 社团成员管理
- 通知信息管理
- 入团申请处理
- 费用管理

### 社团成员后台

- 社团信息管理
- 社团成员管理
- 入团申请
- 社团活动管理
- 通知信息管理
